## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://thomaswoods25/Module-24.-Interactive-MERN-SPA-Project/
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   npm run build
   ```
